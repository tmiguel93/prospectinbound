# Regras de negócio

As regras comerciais são centralizadas na API e configuráveis por produto. Valores monetários são armazenados em centavos inteiros.

Cada produto possui parceiro, planos, uma regra de comissão configurável e um pipeline associado. Um pipeline possui de duas a cinco etapas, exatamente uma inicial e uma final. O seed cria os produtos demonstrativos de Oficinas e Barbearias/Salões com comissão de R$ 30,00 + 10% recorrente.
